# 쇼핑 리스트 앱

순수 HTML/CSS/JavaScript로 만든 쇼핑 리스트 웹 앱입니다.

## 기능

- 아이템 추가 (버튼 클릭 또는 Enter 키)
- 완료/미완료 체크 토글
- 아이템 삭제
- 전체/미완료/완료 필터
- 완료 항목 일괄 삭제
- 통계 표시 (총/완료/미완료 개수)
- localStorage를 통한 데이터 영속성

## 사용법

`shopping-list.html` 파일을 브라우저에서 열면 바로 사용할 수 있습니다.

## 테스트 실행

```bash
npm install
node shopping-list.test.mjs
```
